# BuildX Analytics - Zomato Restaurant Dataset

## Dataset Selected
Zomato Restaurant Dataset - cleaned_dataset.csv

## Why I Selected It
I selected this dataset because it contains real restaurant data with ratings, cost, location, and cuisine type. It helps understand what drives restaurant success in Bangalore.

## Three Business Questions Answered
1. Which restaurants have rating > 4.5? - Asia Kitchen, Byg Brewski, Belgian Waffle Factory etc.
2. Which restaurant type has the most restaurants? -  Microbrewery with 17 restaurants
3. How does price affect rating? - Expensive restaurants have avg rating 3.85 vs Budget 3.17

## Tools Used
- Python / Colab for Data Cleaning
- SQL for querying data
- Power BI Desktop for Dashboard

## One Key Insight
From the dashboard, Expensive restaurants have the highest average rating of 3.85, but Mid-Range ₹300-₹700 offers best value.

## Project Files
- `notebook.ipynb` - Data cleaning code
- `cleaned_dataset.csv` - Cleaned data
- `queries.sql` - 3 SQL queries with results  
- `dashboard.pbix` - Power BI dashboard
